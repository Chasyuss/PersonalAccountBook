# 개인 지출 관리 어플리케이션

## 👨‍🏫 프로젝트 소개
React를 사용하여 만든 **개인 지출 관리 어플리케이션** 입니다.

## 📢 Running
```
#development
~$ yarn

# watch mode
~$ yarn dev

#redux 설치 
~$ yarn add redux react-redux @reduxjs/toolkit
```
## 📌 주요 기능
- 수정 버튼을 누르면, 기존 지출의 데이터를 수정하고 ‘홈’ 으로 이동
- 삭제 버튼 클릭 시  사용자에게 확인받은 뒤 삭제
- 지출 추가
- 그 외
  - **context** 전역관리
  - **redux** 전역관리
